# Pet Shop Website 

A modern and responsive pet shop website designed to provide an intuitive experience for customers looking for pet care services, appointments, and information.

---

##  Features

- Responsive and mobile-friendly layout  
- Clean and modern UI design  
- Service presentation (bath, grooming, boarding, etc.)  
- Appointment/contact sections  
- User-friendly navigation  
- Focus on usability and visual clarity  

---

## Project Purpose

This project was developed as a front-end practice exercise focused on improving skills in UI design, component structuring, and responsive web development. The goal was to simulate a real-world commercial website for a pet shop business.

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (or React if applicable)  
- Responsive Design Principles
